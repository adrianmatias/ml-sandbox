# ragblog

This project perfomrs Retrieval Augmented Generation on from a crawled blog.

## Init

### Install poetry environment
```
sudo apt install pipx
https://python-poetry.org/docs/#installation
poetry shell
```
### Lint
```
sh lint.sh
```
### Test
```
pytest -vv
```
### run
```
cd ragblog
python run.py
```
